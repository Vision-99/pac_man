# 🟡 Pac-Man Console Game

Simple Pac-Man–style game in C++ (Windows only).  
Control Pac-Man using `W`, `A`, `S`, `D` keys — avoid 👻 and collect dots 🍒!

---

## 🎮 Controls

| Key | Action      |
|-----|-------------|
| W   | Move Up ⬆️   |
| A   | Move Left ⬅️ |
| S   | Move Down ⬇️ |
| D   | Move Right ➡️|

---

## ⚙️ Requirements

- Windows OS 
- C++ Compiler (MSVC or MinGW)  
- Console with UTF-8 support

---

## ▶️ Run

```bash
g++ pacman.cpp -o pacman
pacman
